## 2. 角色系统 (Character System)

### 2.1 随机开局 (Roguelike)
游戏开始时，系统随机生成5名角色供玩家选择（N选1）。

#### 属性维度

**学校背景** (决定初始属性范围、隐藏智商系数、简历通过率基础值)：
- Top2：初始属性极高，隐藏智商极高
- 985：初始属性高，隐藏智商高
- 211：初始属性中等
- 双非：初始属性一般
- 民办：初始属性较低

**家庭背景** (提供特殊Buff及资源支持)：
- 富二代：心态恢复速度 +50%
- 中产家庭：初始软技能 +10
- 互联网世家：大三时有30%概率触发"亲戚内推"事件
- 工薪阶层：精力上限 +20（由100变为120，更耐肝）

**显性四维属性** (玩家可见，核心养成指标)：
- **学业 (GPA)**：初始3.5 (Max 4.0)。决定保研资格、考研复试印象分以及是否延毕
- **项目能力 (Project)**：0-999。决定简历筛选率，应对面试中的"实战/场景题"
- **八股分 (Knowledge)**：0-999。决定技术面通过率，应对面试中的"原理/基础题"
- **软技能 (SoftSkill)**：0-999。决定HR面通过率、薪资谈判及部分特殊事件判定

**隐藏属性** (玩家不可见)：
- **学习能力/智商 (Learning Ability/IQ)**：
  - 来源：根据学校背景和家庭背景综合随机生成
  - 作用：作为"经验倍率"系数。智商越高的角色，进行"刷题"、"做项目"等行动时，获得的属性点加成越高（例如：普通人刷题+8，天才刷题+12）

## 3. 核心循环与时间轴 (Core Loop & Timeline)

保持 v1.0 逻辑，但相关判定调整为适配新的四维属性。

### 3.1 时间系统
总时长：48个月（4年）
推进方式：手动结束（+1月）或 事件跳过（实习+3月）

### 3.2 阶段划分
- 大一/大二 (Month 1-24) - 积累期：目标刷GPA，积累基础属性
- 大三 (Month 25-36) - 实习期：找实习，丰富简历
- 大四 (Month 37-48) - 抉择期：秋招冲刺 或 考研上岸

### 3.3 资源系统
- 精力：初始100，工薪120
- 心态：初始80
