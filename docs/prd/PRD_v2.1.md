- 版本号：v2.1 (Mental Health & Realism Update)
- 文档状态：迭代增量
- 关联文档：本科生模拟器_PRD_v2.0_Final.md
- 核心变更：心态崩溃机制、过劳死定义、精力/季度结算明确、实名高校显示

---

## 1. 心态与健康系统重构

### 1.1 心态归零机制
- 触发：行动或季度结算后，心态 ≤ 0。
- 惩罚：
  - 强制住院：跳过下一个季度（例如 Q3 崩溃 -> Q4 住院 -> 游戏进入 Q5）。
  - 属性受损：GPA/项目/八股/软技各 -10（GPA -0.1）。
  - 永久 Debuff：获得【抑郁】，心态上限降至 80。

### 1.2 过劳死结局
- 判定：累计触发 2 次以上心态崩溃。
- 结果：游戏立即结束，进入结局【过劳死/因病离世】。

### 1.3 行动代价调整
- 包装项目：精力 50、GPA -0.3、心态 -20（原为 -0）。
- 理由：大项目属于高强度脑力劳动，应带来显著心态消耗。

---

## 2. 休息与季度结算逻辑

### 2.1 季度收尾流程
- “结束本季”按钮不再触发具体行动，而是打开【季度结算面板】。
- 玩家必须四选一：
  1. 宿舍摆烂（0 元，心态 +10）
  2. 聚餐娱乐（200 元，心态 +40）
  3. 特种兵旅游（1500 元，心态 +80）
  4. 奢靡生活（5000 元，心态回满，富二代限定）
- 结束方式将决定是否计入“本季度有娱乐消费”。

### 2.2 精力恢复说明
- 精力不可跨季度累积。
- 一旦进入新的季度（无论上一季如何结束），精力自动重置为当前精力上限。

---

## 3. 代入感增强

### 3.1 实名高校显示
- 角色卡与主界面均显示随机抽取的真实高校名称，并附加所属层级（如“电子科技大学 (985)”）。
- 目的：强化身份认知和代入感，与 v2.0 的实名高校库呼应。

---

## 交付指标
1. 心态 ≤ 0 时立即触发惩罚，强制跳过下一季度并附加属性、心态上限 Debuff。
2. 心态崩溃累计 2 次直接结束游戏，给出【过劳死】结局。
3. “结束本季”弹出结算面板，四个选项之一才能进入下一季度。
4. 包装项目行动更新为心态 -20。
5. 角色选择及 UI 身份区域显示真实高校名称。
