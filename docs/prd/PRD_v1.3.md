# 产品需求文档 (PRD) - 《本科生模拟器：四年之约》

## 项目名称
本科生模拟器：四年之约 (Undergraduate Simulator)

## 版本号
v1.3 (Replaced)

## 核心变更
- GPA惩罚激进化（实习-0.8/季）
- 娱乐消费调整（聚餐200→150，旅游3000→2500）
- 新增兼职打工行动

---

## 1. 产品概述 (Product Overview)

### 1.1 背景与目标
基于当前互联网求职环境与大学生现状，开发一款具有黑色幽默风格的文字模拟养成游戏。

核心体验目标：让玩家在有限的时间、精力与金钱资源下，在"保研内卷"、"大厂实习"、"考研上岸"三条路线中做出艰难抉择，体验从大一入学到毕业的四年真实历程。

### 1.2 核心玩法
- **Roguelike开局**：随机出身决定天赋上限与初始资金
- **资源管理**：精力、心态与金钱的三角平衡
- **简历养成**：可视化的简历填充过程
- **实习抉择**：在薪资、距离、职业发展中做权衡

---

## 2. 角色系统 (Character System)

### 2.1 随机开局 (Roguelike)
游戏开始时，系统随机生成5名角色供玩家选择（N选1）。

#### A. 出身背景 (决定初始盘面)

**学校背景** (决定初始属性基数、简历筛选权重)：
- Top2：自带光环，简历免筛选
- 985：大厂敲门砖
- 211：中规中矩
- 双非：简历筛选有惩罚系数
- 民办：地狱模式，需极高属性才能逆袭

**家庭背景** (决定资源与Buff)：
- 富二代：心态恢复速度 +50%，初始资金 50,000 + 月生活费 8,000
- 中产家庭：初始软技能 +10，初始资金 5,000 + 月生活费 2,500
- 互联网世家：大三时有30%概率触发"亲戚内推"事件，资金适中
- 工薪阶层：精力上限 +20，初始资金 1,000 + 月生活费 1,200 (生存压力极大，需兼职或极度省吃俭用)

#### B. 显性四维属性 (玩家可见)
- **学业 (GPA)**：0.0-4.0。决定保研/考研/延毕
- **项目能力 (Project)**：0-999。决定简历筛选及实战面试
- **八股分 (Knowledge)**：0-999。决定技术面试
- **软技能 (SoftSkill)**：0-999。决定HR面试及薪资谈判

#### C. 隐藏属性 (玩家不可见)
- **学习能力/智商 (IQ)**：经验倍率系数

## 3. 核心循环与时间轴 (Timeline & Loop)

### 3.1 资源系统

**精力 (Energy)**：行动点数。不再能在当月回复。所有回复精力的操作意味着"休息一整晚/几天"，直接进入下一月。

**心态 (Sanity)**：生命值。
- 自然衰减：若当月未进行任何娱乐消费，心态额外 -10 (枯燥惩罚)
- 归零惩罚：触发【崩溃/住院】，强制跳过1-2个月，心态上限永久锁定80

**金钱 (Money)**：生存核心。
- 基础消耗：每月自动扣除 800元 (饭钱/网费等刚需)
- 额外消耗：租房、娱乐
- 破产：金钱 < 0 时强制结束实习或触发"向家里要钱/网贷"事件（大幅扣心态/背负债务）

### 3.2 阶段划分
- **大一/大二 (蛰伏期)**：刷GPA，积累属性，省钱/兼职
- **大三 (实习黄金期)**：找实习，权衡通勤/租房成本，积攒大厂履历
- **大四 (决战期)**：秋招冲刺 或 考研上岸

## 4. 功能模块详解

### 4.1 行动模块 (Action Center)

玩家每月的操作面板。关键变更：大幅增加非学习类行动对GPA的惩罚。

| 行动名称 | 消耗资源 | 效果 | 备注 |
|---------|---------|------|------|
| 认真上课 | 20精力 | GPA +0.1 | 基础操作，唯一的GPA来源 |
| 包装项目 | 30精力 | 项目能力 +8，GPA -0.2 | (惩罚加倍) 沉迷做项目，期末突击复习都来不及，绩点血崩 |
| 刷题/背八股 | 20精力 | 八股分 +8，心态 -5 | 枯燥操作 |
| 兼职打工 | 30精力 | 金钱 +500，GPA -0.2，心态 -10 | (惩罚加倍) 穷人的恶性循环，为了生活费牺牲学业 |
| 【结算】宿舍死宅 | 0金钱 | 进入下一月，精力回满，心态 +5 | 效果微弱，长期死宅会导致心态自然衰减 |
| 【结算】聚餐娱乐 | 200金钱 | 进入下一月，精力回满，心态 +30 | (费用下调) 更合理的社交成本，性价比最高的回复方式 |
| 【结算】豪华旅游 | 3000金钱 | 进入下一月，精力回满，心态 +80 | 彻底放松，只有富哥或拿到大厂工资才敢消费 |

### 4.2 实习招聘与薪资 (Internship & Salary)

#### A. 薪资体系 (浮动制)
最终日薪 = 基础薪资区间(随机) + 岗位修正 + 软技能谈判加成

| 公司分级 (Tier) | 基础薪资范围 (RMB/天) | 特点描述 |
|----------------|---------------------|---------|
| Tier 3 (小厂/作坊) | 80 - 120 | 仅够覆盖吃饭，必须倒贴房租 |
| Tier 2 (中厂/独角兽) | 150 - 250 | 能覆盖生活费，略有结余 |
| Tier 1 (互联网大厂) | 300 - 600 | 包含房补。高薪是这游戏里唯一能让你"豪华旅游"的资本 |

#### B. 地理与通勤机制
- **同城-近距离 (40%)**：无额外消耗
- **同城-远距离 (30%)**：通勤折磨。每月精力上限 -20，心态额外 -10
- **异地 (30%)**：必须租房。每月强制扣除房租 (1500-3500元)

#### C. 实习反馈循环 (高风险高回报)
获得Offer -> 选择【去实习】 -> 时间+3个月 -> 每月结算金钱/心态 (可能遭遇PUA扣心态) -> GPA -0.8 (惩罚剧增，3个月完全不上课，不仅没平时分，期末也大概率挂科) -> 获得【大厂履历】

### 4.3 面试战斗 (Interview Battle)
- 判定公式：玩家属性 >= 题目难度 + (公司Tier * 10)
- 薪资谈判：若软技能 > 80，面试通过后可触发谈判环节

## 5. 结局判定系统 (Endings)

游戏在第48个月结束时进行结算：

- **【坏结局：劝退/延毕】** (GPA < 2.0)。由于实习扣分加重，此结局触发概率大幅提升
- **【研究生上岸】** (考研分达标，分985/本校)
- **【就业结局】**
  - 大厂SSP：持有Tier 1 Offer且数量>=2，且薪资处于高位区间
  - 社畜预备役：持有任意Offer
- **【保研】** (GPA > 3.8)
- **【普通坏结局：家里蹲】**
