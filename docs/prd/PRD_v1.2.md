## 1. 产品概述 (Product Overview)

### 1.1 背景与目标
基于当前互联网求职环境与大学生现状，开发一款具有黑色幽默风格的文字模拟养成游戏。

核心体验目标：让玩家在有限的时间、精力与金钱资源下，在"保研内卷"、"大厂实习"、"考研上岸"三条路线中做出艰难抉择，体验从大一入学到毕业的四年真实历程。

### 1.2 核心玩法
- **Roguelike开局**：随机出身决定天赋上限与初始资金
- **资源管理**：精力、心态与金钱的三角平衡
- **简历养成**：可视化的简历填充过程
- **实习抉择**：在薪资、距离、职业发展中做权衡

## 2. 角色系统 (Character System)

### 2.1 随机开局 (Roguelike)
游戏开始时，系统随机生成5名角色供玩家选择（N选1）。

#### A. 出身背景 (决定初始盘面)

**学校背景** (决定初始属性基数、简历筛选权重)：
- Top2：自带光环，简历免筛选
- 985：大厂敲门砖
- 211：中规中矩
- 双非：简历筛选有惩罚系数
- 民办：地狱模式，需极高属性才能逆袭

**家庭背景** (决定资源与Buff)：
| 家庭类型 | 初始资金 | 月生活费 | 其他Buff |
|---------|---------|---------|---------|
| 富二代 | 50,000 | 5,000 | 心态恢复+50% |
| 中产家庭 | 适中 | - | 软技能+10 |
| 互联网世家 | - | - | 大三30%内推概率 |
| 工薪阶层 | 2,000 | 1,500 | 精力+20 |

#### B. 显性四维属性 (玩家可见)
- **学业 (GPA)**：0.0-4.0。决定保研/考研/延毕
- **项目能力 (Project)**：0-999。决定简历筛选及实战面试
- **八股分 (Knowledge)**：0-999。决定技术面试
- **软技能 (SoftSkill)**：0-999。决定HR面试及薪资谈判

#### C. 隐藏属性 (玩家不可见)
- **学习能力/智商 (IQ)**：经验倍率系数

## 3. 核心循环与时间轴 (Timeline & Loop)

### 3.1 资源系统 (Updated)

**精力 (Energy)**：行动点数。归零强制结束本月

**心态 (Sanity)**：生命值
- 自然衰减：大一/二 (-2/月)，大三/四 (-5/月)
- 归零惩罚：触发【崩溃/住院】，强制跳过1-2个月，心态上限永久锁定80

**金钱 (Money)**：新增核心资源
- 用途：租房（异地实习）、高阶娱乐（回心态）、日常开销
- 破产：金钱 < 0 时强制结束实习或触发"向家里要钱"羞耻事件（大幅扣心态）

### 3.2 阶段划分
- **大一/大二 (蛰伏期)**：刷GPA，积累属性，省钱/兼职
- **大三 (实习黄金期)**：找实习，权衡通勤/租房成本，积攒大厂履历
- **大四 (决战期)**：秋招冲刺 或 考研上岸

## 4. 功能模块详解

### 4.1 行动模块 (Action Center)

玩家每月的操作面板。

| 行动名称 | 消耗资源 | 效果 | 备注 |
|---------|---------|------|------|
| 认真上课 | 20精力 | GPA +0.1 | 基础操作 |
| 包装项目 | 30精力 | 项目能力 +8，简历增加记录 | 核心养成 |
| 刷题/背八股 | 20精力 | 八股分 +8，心态 -5 | 枯燥操作 |
| 宿舍睡觉 | 0金钱 | 精力回满，心态 +10 | 低成本恢复，仅够维持生存 |
| 普通娱乐 | 200金钱 | 精力回满，心态 +30 | 花钱买快乐 |
| 豪华旅游 | 2000金钱 | 精力回满，心态 +100 | 富二代或高薪后的奖励 |

### 4.2 实习招聘与薪资 (Internship & Salary) - 重构

#### A. 薪资体系 (去固定化)
最终日薪 = 基础薪资区间(随机) + 岗位修正 + 软技能谈判加成

| 公司分级 (Tier) | 基础薪资范围 (RMB/天) | 特点描述 |
|----------------|---------------------|---------|
| Tier 3 (小厂/作坊) | 50 - 100 | 仅够饭钱，甚至"用爱发电" |
| Tier 2 (中厂/独角兽) | 100 - 180 | 行业标准，但也存在压价 |
| Tier 1 (互联网大厂) | 200 - 500 | 跨度极大。200白菜价，500 SSP核心岗 |

#### B. 地理与通勤机制
生成Offer时随机附加地理标签，迫使玩家在金钱与心态间做取舍：

| 标签 | 概率 | 影响 |
|------|-----|------|
| 同城-近距离 | 40% | 无额外消耗 |
| 同城-远距离 | 30% | 每月精力上限 -10，心态额外 -5 |
| 异地 | 30% | 必须租房(2000-4000元/月)。金钱不足时开启【省吃俭用】 |

#### C. 实习反馈循环
获得Offer -> 选择【去实习】 -> 时间+3个月 -> 每月结算金钱/心态 (可能遭遇PUA扣心态) -> GPA -0.2 -> 获得【大厂履历】

### 4.3 面试战斗 (Interview Battle)
- 判定公式：玩家属性 >= 题目难度 + (公司Tier * 10)
- 薪资谈判：若软技能 > 80，面试通过后可触发谈判环节，成功则薪资 +10%~20%

## 5. 结局判定系统 (Endings)

游戏在第48个月结束时进行结算：

- **【坏结局：劝退/延毕】** (GPA < 2.0)
- **【研究生上岸】** (考研分达标，分985/本校)
- **【就业结局】**
  - 大厂SSP：持有Tier 1 Offer且数量>=2，且薪资处于高位区间
  - 社畜预备役：持有任意Offer
- **【保研】** (GPA > 3.8)
- **【普通坏结局：家里蹲】**
